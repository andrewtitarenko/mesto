# **Место**
Одностраничный проект-сайт об интересных местах для путешествия по России

## **В проекте используется:** 

- флексбокс-верстка; 
- гриды
- адаптивная верстка; 
- файловая структура по правилам Nested БЭМ.
- JavaScript
